## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Rust panic → CPython | an unexpected panic must not unwind into the interpreter as an abort |
| backend string → Backend | a caller-supplied backend name selects a compute path; un-built/device-absent must fail typed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-11 | Denial of Service | Rust panic unwinding into CPython → UB/abort | mitigate | pyo3 0.28 auto `catch_unwind` → `PanicException`; the `guard` remap → `TreeliteError` (D-07), interpreter never aborts |
| T-08-12 | Spoofing | device-absent GPU backend → silent wrong result | mitigate | `CubeclError::DeviceUnavailable` is a typed error → `TreeliteError`, NEVER a silent CPU fallback (D-08, Phase 7 carry-forward) |
| T-08-13 | Tampering | un-built backend name requested | mitigate | the `other =>` match arm returns a typed `TreeliteError` naming the wheel's BUILT_BACKENDS — no UB, no fallback |
</threat_model>
