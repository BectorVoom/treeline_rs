## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| packed nodes buffer → decode | The HistGB `nodes` byte buffer is untrusted; itemsize, buffer length, and every index must be validated before reads |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-18 | Tampering / DoS | `expected_sizeof_node_struct` not in {52,56} or `nodes` buffer shorter than `node_count × itemsize` → OOB read | mitigate | Reject any itemsize not in {52,56} (upstream TREELITE_LOG(FATAL), sklearn.cc:366) and verify buffer length ≥ `node_count × itemsize` with a typed `SklError::HistGbDecode` BEFORE any field decode |
| T-04-19 | Denial of Service | `feature_idx` indexing `features_map`/`categories_map` out of range | mitigate | Bounds-check `feature_idx` against `features_map.len()` (and `categories_map` length) before access; typed error, never OOB |
| T-04-20 | Denial of Service | `bitset_idx` indexing `left_cat_bitmap` out of range (`8*bitset_idx + i/32`) | mitigate | Bounds-check `8*bitset_idx + 7 < bitmap.len()` before the 256-category scan; typed error |
| T-04-21 | Tampering | Native-endian transmute of the packed struct (alignment/endianness UB) | mitigate | Decode field-by-field with `from_le_bytes`; NO `transmute`/`bytemuck` (Phase-3 D-08 ban) — enforced by a grep gate in acceptance criteria |
| T-04-SC | Tampering | npm/pip/cargo installs | mitigate | No new third-party packages; `treelite-sklearn` is internal — N/A |
</threat_model>
